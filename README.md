# ABC AI 🐻

Agent Bear Corps - Multi-provider AI Agent Runtime

## Features

- **Multi-Provider Support**: Anthropic, OpenAI, Google, xAI, DeepSeek, Alibaba, Moonshot, Baidu
- **Multi-Persona**: Professional, Friendly, Technical, Teacher, Creative, Humorous
- **Persistent Memory**: SQLite-based conversation history
- **HTTP API**: RESTful API for external integration
- **Setup Wizard**: Browser-based configuration

## Quick Start

### Option 1: Setup Wizard (Recommended)

```bash
# Clone repo
git clone https://github.com/jpmoregain-eth/abc-ai.git
cd abc-ai

# Install dependencies
pip install -r requirements.txt

# Run setup wizard (opens browser)
python setup.py
```

The setup wizard will:
1. Open your browser at `http://localhost:8080`
2. Guide you through 6 configuration steps
3. Generate your agent config file
4. Optionally launch the agent immediately

### Option 2: Use Existing Config

```bash
# If you already have a config from abc-setup
python agentbear.py chat --config my-agent.yaml

# Or start API server
python agentbear.py api --config my-agent.yaml --port 5000
```

## Usage

### Interactive Chat
```bash
python agentbear.py chat --config my-agent.yaml
```

### Single Message
```bash
python agentbear.py chat -m "Hello!" --config my-agent.yaml
```

### Start API Server
```bash
python agentbear.py api --port 5000 --config my-agent.yaml
```

### Show Agent Info
```bash
python agentbear.py info --config my-agent.yaml
```

## API Endpoints

When running `python agentbear.py api`:

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Health check |
| `/api/info` | GET | Agent information |
| `/api/chat` | POST | Send message to agent |
| `/api/memory/<session_id>` | GET | Get conversation history |
| `/api/memory/<session_id>` | DELETE | Clear session memory |
| `/api/model` | GET | Get current model |
| `/api/model` | POST | Switch model |

### Chat Example
```bash
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!", "session_id": "test-001"}'
```

Response:
```json
{
  "response": "Hello! I'm your AI agent...",
  "session_id": "test-001",
  "model": "claude-sonnet-4-6",
  "personas": ["professional", "humorous"]
}
```

## Configuration

Config files are YAML format (generated by setup wizard or abc-setup):

```yaml
agent:
  name: my-agent
  owner: Derek
  personas:
    - professional
    - humorous
  language: en

models:
  primary: claude-sonnet-4-6
  providers:
    anthropic:
      api_key: "sk-ant-..."
    openai:
      api_key: "sk-..."
```
