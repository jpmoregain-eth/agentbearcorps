# ABC AI 🐻

Agent Bear Corps - Multi-provider AI Agent Runtime

## Features

- **Multi-Provider Support**: Anthropic, OpenAI, Google, xAI, DeepSeek, Alibaba, Moonshot, Baidu
- **Multi-Persona**: Professional, Friendly, Technical, Teacher, Creative, Humorous
- **Persistent Memory**: SQLite-based conversation history
- **HTTP API**: RESTful API for external integration
- **Model Switching**: Runtime model switching with fallback support

## Quick Start

1. **Install dependencies**:
```bash
pip install -r requirements.txt
```

2. **Copy example config**:
```bash
cp agent_config.example.yaml agent_config.yaml
```

3. **Edit config and add API keys**:
```yaml
models:
  providers:
    anthropic:
      api_key: "sk-ant-..."
```

4. **Run the agent**:
```bash
# Interactive mode
python agent.py --config agent_config.yaml

# API server
python api_server.py --config agent_config.yaml --port 5000
```

## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Health check |
| `/api/info` | GET | Agent information |
| `/api/chat` | POST | Send message to agent |
| `/api/memory/<session_id>` | GET | Get conversation history |
| `/api/memory/<session_id>` | DELETE | Clear session memory |
| `/api/model` | GET | Get current model |
| `/api/model` | POST | Switch model |
| `/api/config` | GET | Get configuration |
| `/api/config` | POST | Update configuration |

### Chat Example

```bash
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello! What can you do?",
    "session_id": "test-session-001"
  }'
```

Response:
```json
{
  "response": "Hello! I'm your AI agent...",
  "session_id": "test-session-001",
  "model": "claude-sonnet-4-6",
  "personas": ["professional", "humorous"]
}
```
